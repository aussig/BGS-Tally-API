# BGS-Tally API

BGS-Tally is a tool to track and report your Background Simulation (BGS) and Thargoid War (TW) activity in Elite Dangerous. The main project is [here on GitHub](https://github.com/aussig/BGS-Tally).

BGS-Tally is able to connect to remote servers to push users' contributions using an API and this project defines the structures and data used for data transfer, as an OpenAPI YAML file.
